캐스케이드 규칙

  1. 중요도(!IMPORTANT)&출처
    • 기본적으로 !IMPORTANT로 선언된 모든 규칙은 그렇지 않은 규칙보다 우선한다.
    • 출처는 제작자, 사용자, 사용자 에이전트(User Agent)로 구분한다.

      • 우선순위

      5. 사용자 에이전트
      4. 사용자 스타일
      3. 제작자 스타일
      2. 제작자 !IMPORTANT스타일
      1. 사용자 !IMPORTANT스타일

  2. 구체성
    구체성값이 높은것이 우선된다.

  3. 선언순서
    나중에 선언된 규칙일수록 우선된다.
