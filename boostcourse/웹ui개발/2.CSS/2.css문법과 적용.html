CSS문법

    h1 { color: yellow; font-size:2em; }

  • 선택자(selector) - "h1"
  • 속성(property) - "color"
  • 값(value) - "yellow"
  • 선언(declaration) - "color: yellow", "font-size: 2em"
  • 선언부(declaration block) - "{ color: yellow; font-size:2em; }"
  • 규칙(rule set) - "h1 { color: yellow; font-size:2em; }"

    • 선택자와 선언부 사이, 선언과 선언 사이는 앞뒤로 개행을 해도 상관이 없다.
    하지만 속성이름과 속성값 사이에는 개행을 하면 안된다.

    CSS의 속성은(Property), HTML의 속성은(Attribute)

주석(comment tags)
/* 내용 */


inline방식
해당 요소에 직접 스타일속성을 이용해서 규칙들을 선언하는 방법(자주사용안함)

internal방식
문서에 style태그를 이용하는 방법(헤드태그 내부에 들어감)

external방식
외부 스타일 시트파일을 이용하는 방법(주로 사용되는 방법)
head 내부에 link를 선언한 후 href 속성을 이용해 CSS 파일의 경로를 입력한다.
rel 속성은 연결되는 파일이 문서와 어떤 관계인지를 명시하는 속성으로, CSS 파일은 'stylesheet' 라고 적어야 한다.

@import방식
스타일 시트 내에서 다른 스타일 시트 파일을 불러오는 방식(거의 쓰이지 않음)
