z-index 속성

요소가 겹치는 순서(쌓임 순서 또는 stack order)를 지정하는 속성

기본 값 : auto

auto : 쌓임 순서를 부모와 동일하게 설정(기본값)
number : 해당 수치로 쌓임 순서를 설정(음수 허용)

  • position 값이 static이 아닌 경우 지정가능
  • 순서 값이 없을 경우 생성순서(코드상 순서)에 따라 쌓임
  • 부모가 z-index 값이 있을 경우 부모 안에서만 의미있음
  • 큰 값이 가장 위쪽(음수 사용 가능)
