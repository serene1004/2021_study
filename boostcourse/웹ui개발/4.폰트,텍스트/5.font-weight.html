font-weight 속성

글꼴의 굵기를 지정하는 속성으로 기본값 : normal

font-weight: normal | bold | bolder | lighter | number | initial | inherit ;

normal : 기본 값 (400)
bold : 굵게 표현(700)
bolder : 부모 요소 보다 두껍게 표현
lighter : 부모 요소 보다 얇게 표현
number : 100, 200, 300, 400, 500, 600, 700, 800, 900 (클수록 더 두껍게 표현)

실무에서는 normal과 bold를 많이 사용함
부모 요소에 영향이 있는 bolder과 lighter는 사용을 될 수 있으면 지양한다(사용할땐 신중하게)
