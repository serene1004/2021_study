text-decoration 속성

텍스트의 장식을 지정하는 속성으로 기본값은 차례대로 3가지 속성의 값이다

기본 값 : none currentColor solid

text-decoration: text-decoration-line text-decoration-color text-decoration-style | initial | inherit;

  • text-decoration-line : 텍스트 꾸밈의 종류를 지정하는 속성
    기본 값 : none

    • none : 텍스트 꾸밈을 생성하지 않음 (기본값)
    underline : 밑줄로 꾸밈을 설정
    overline : 윗줄로 꾸밈을 설정
    line-through : 중간을 지나는 줄로 꾸밈을 설정

  • text-decoration-color : 텍스트 꾸밈의 색상을 지정하는 속성
    기본 값 : currentColo
  • text-decoration-style : 꾸밈에 사용되는 선의 스타일을 지정하는 속성
    기본 값 : solid

    • solid : 한줄 스타일 (기본 값)
    double : 이중선 스타일
    dotted : 점선 스타일
    dashed : 파선 스타일
    wavy : 물결 스타일

일반 경우

text-decoration: overline;

text-decoration: underline;

text-decoration: line-through;

부모 내 자식요소가 float될 경우 상속이 해제됨

text-decoration: overline;

부모 내 자식요소가 absolute 경우 상속이 해제됨

text-decoration: overline;
