프로필 사진 업로드 :



type="file/submit/reset/image/button"

  • file : 파일을 서버에 올릴때 사용하는 버튼
    브라우저에 따라 표현되는 형태는 조금씩 다르지만, 모두 같은 역할을 한다.
  • submit : form의 값을 전송하는 버튼
  • reset : form의 값을 초기화하는 버튼
  • image : 이미지를 삽입할 수 있는 버튼 (submit과 동작이 동일함)
  • button : 아무 기능이 없는 버튼
