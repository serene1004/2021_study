기본정보

성별 :
부가정보 취미 : 영화감상 음악감상 독서

label은 form요소의 이름과 form 요소를 연결시켜주기 위함으로 웹 접근성 향상에 도움이 된다.(필수적)
모든 form 요소에 사용할 수 있다.
form 요소의 id 속성값과 label의 for 속성값을 같게 적어주어야 합니다.

fieldset : 여러 개의 폼 요소를 그룹화하여 구조적으로 만들기 위해 사용

legned : 폼 요소의 제목으로 fieldset 요소 내부에 작성

form은 form요소들을 감싸는 태그로 데이터를 그룹화하여 서버에 전송하는 역할을 하는 태그이다.
만약 fieldset으로 구조화 되어있다면 fieldset도 함께 감싸는 역할을 한다.

action : 폼 데이터를 처리하기 위한 서버의 주소

method : 데이터를 전송하는 방식을 지정(get, post)

get 방식은 데이터가 전송될 때 주소창에 데이터가 노출이 된다.

post 방식은 데이터가 전송될 때 데이터가 노출되지 않는다.
