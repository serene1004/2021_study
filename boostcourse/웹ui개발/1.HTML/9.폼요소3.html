select는 선택목록상자 또는 콤보박스라 하며, 몇개의 선택지를 리스트형태로 노출하고 그중 하나를 선택할수 있게 하는 태그이다.(multiple 속성을 사용하면 다중 선택도 가능)
select내부의 option으로 각 항목을 나타냄. option의 속성으로 selected가 있으며 이는 선택된 항목을 의미한단.

textarea는 한 줄만 입력할 후 있는 input=type"text"와 달리 여러줄의 텍스트를 입력할때 사용가능하다.
textarea에는 텍스트 상자의 크기를 조절하는 rows, cols 속성이 있다.
cols : 가로 크기를 조절하는 속성
rows : 세로 크기를 조절하는 속성

button은 버튼을 만들 때 사용하며 submit, reset, button 3가지의 타입이 있다.
각 버튼은 input타입의 submit, reset, button과 모두 같은 기능을 가진 버튼이다.
다만, 빈 태그가 아니며 내용을 안에 직접 넣을 수 있으므로 좀 더 자유로운 스타일 표현이 가능하다.
